Ich wollte, dass meine App mit mir spricht.

Es begann mit einem flachen WhatsApp-Diktat — ein Block ohne Absätze. Am Ende des Abends hielt ich eine Taste, sprach, und eine Stimme antwortete mir. Komplett lokal. Ohne Abo. Ohne Cloud.

fn diktiert · ⌥ spricht whisper.cpp · Ollama · Piper M1 Pro · 16 GB · 0 €/Monat

01 Der Auslöser

Ein Diktat, das gut erkannt war — und trotzdem schade.

Ich hatte einen längeren Text in WhatsApp gesprochen. Murmel hat jedes Wort richtig erkannt. Aber es war ein einziger, langer Block — keine Absätze, keine Struktur. Beim Lesen merkt man: gesprochene Gedanken brauchen Gliederung, sonst gehen sie unter.

Die Erkennung stimmte. Die Form fehlte.

Daraus wurde der erste Baustein des Abends: ein Modus „Strukturiert", der diktierten Fließtext in sinnvolle Absätze gliedert — wortgetreu, ohne umzuschreiben. In Messengern greift er automatisch.

02 Die Vision

Eine Taste, die alles Relevante steuert.

Aus „mach den Text schöner" wurde schnell eine größere Frage: Wenn Murmel schon hört, versteht und schreibt — warum dann nur tippen? Warum nicht über die fn-Taste alles Relevante am Rechner steuern und Antworten bekommen?

Der ehrliche Befund zuerst: Das Fundament steht längst. Murmel hört (whisper), denkt (Ollama) und antwortet (TTS). Was fehlt, ist nicht die Möglichkeit — es ist ein Intent-Router und eine Aktions-Schicht. Daraus wurde eine Architektur in fünf Bausteinen.

① Router

Verstehen

Diktieren, fragen oder befehlen? Erst Heuristik, dann kleines Modell.

② Werkzeuge

Handeln

Apps, System, Dateien, n8n, Web — eine Registry sicherer Tools.

③ Daten

Wissen

RAG über eigene Dateien + Live-Kontext der aktiven App.

④ Modell

Eskalieren

Lokal als Standard, Cloud nur an der Spitze. On-demand geladen.

⑤ Sicherheit

Bestätigen

Kritische Aktionen erst nach Rückfrage. Allowlist statt Vollmacht.

Prinzip

Sparsam

So wenig wie möglich, so viel wie nötig — Vollgas nur, wenn die Taste gedrückt ist.

03 Die ehrliche Grenze

16 Gigabyte entscheiden, was realistisch ist.

Beim Feintuning zeigte sich live: Das kleine 3B-Modell kann wortgetreu UND reich formatieren nur halb — es erfindet vereinzelt Wörter. Genau das ist der Flaschenhals jedes lokalen Assistenten: nicht Geld, nicht Möglichkeit, sondern Modellstärke gegen verfügbaren Speicher.

Info-Quelle

Hybrid — lokal als Standard, Cloud optional an der Spitze

Standard-Modell

qwen2.5:7B auf M1 Pro / 16 GB; 14B & Claude nur Spitze

Formatieren

Wortgetreu — Rhetorik durch Typografie, kein Umschreiben

Auslastung

An die fn-Taste gekoppelt — sonst Leerlauf

04 Eine Stimme

Von „fürchterlich" zu Thorsten.

Die erste eingebaute Mac-Stimme war ehrlich gesagt fürchterlich. Die Frage war klar: eine wirklich natürliche Stimme — aber lokal und gratis. Die Antwort heißt Piper, ein neuronales Text-to-Speech, das offline auf dem M1 läuft. Stimme: Thorsten, deutsch, ~60 MB.

Der Weg dahin war nicht gerade: Die vorgebaute macOS-Binary war kaputt (fehlende Bibliotheken), der Release-Tag stimmte nicht (404). Der saubere Weg führte über eine isolierte Python-Umgebung — und dann sprach die App.

~/.claude/tts — Thorsten testen
# neuronale Stimme, komplett offline
echo "Hallo Alex, ich laufe lokal auf deinem Mac." \
  | python -m piper -m de_DE-thorsten-medium.onnx -f out.wav
afplay out.wav   # → Thorsten spricht ✓

05 Claude redet zurück

Der Terminal lernt sprechen.

Ein kleiner Hook schließt den ersten Kreis: Wenn der Assistent im Terminal fertig geantwortet hat, liest Thorsten eine kurze Fassung vor. Diktieren per fn rein, Stimme zurück — ein Mini-Vorgeschmack auf das, was Murmel selbst können sollte. Stummschalten ist ein Handgriff.

Stop-Hook → Sprachausgabe
# an / aus — ein Handgriff
touch ~/.claude/tts/mute   # still
rm    ~/.claude/tts/mute   # wieder gesprochen

06 Der Gesprächs-Modus

fn diktiert. ⌥ unterhält sich.

Der Kern des Abends: eine zweite Taste mit eigener Aufgabe. Rechte ⌥ halten → sprechen → loslassen → Murmel antwortet gesprochen. Kein Text wird eingefügt, der Dialog hat ein Gedächtnis. Zwei neue Bausteine, sauber getrennt:

Hirn

ConversationEngine

Ollama-Chat mit rollendem Verlauf. Antwortet kurz, gesprochen. Fällt 7B → 3B zurück, wenn nötig.

Stimme

PiperSpeaker

Thorsten lokal, neuronal. Fallback auf die System-Stimme — Murmel ist immer sprechfähig.

Taste

Zweiter Tap

Rechte ⌥ parallel zu fn. fn bleibt unverändert Diktat.
